Zend_Gdata_Gapps_MemberEntry

Description
Description | Vars (details) | Methods (details)

Data model class for a Google Apps Member Entry.

Each member entry describes a single member within a Google Apps hosted domain.

To transfer member entries to and from the Google Apps servers, including creating new entries, refer to the Google Apps service class, Zend_Gdata_Gapps.

This class represents <atom:entry> in the Google Data protocol.
